# RareDiseaseApp
A Garmin-based app to help individuals with hEDS track symptoms and convert those inputs, along with collected biometrics, into a simple health score. 

Prereqs for utilizing the Garmin Interface:
1) Have SDK downloaded to your computer: https://developer.garmin.com/connect-iq/sdk/
Follow the instructions depending on the version and device
2) Install the VSCode Monkey C extension (this is step 2 listed in the link above)
3) With both downloaded, make sure to download an API from the SDK to ensure connection
This should allow you to work with the Monkey C Language.

To use app:
1) Make sure you are opening the folder 'app_design', it is in frontend > app_design
2) Open up one of the source files, there labeled under source > '.mc'
3) To check the simulation (working capabilities of watch), go to Run > Run without debugging
This is located at the top of your window bar (next to Go and Terminal)
